# 🧮 Calculator

A simple and responsive web-based calculator built with **HTML**, **CSS**, and **JavaScript**.  
Created as part of the **COS 104** assignment.

---

## 🔍 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧼 Clear/reset function  
- 📱 Responsive design  
- 🎨 Clean and user-friendly UI
